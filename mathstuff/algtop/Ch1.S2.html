2.  The Fundamental Group

A path starting and ending at a point is said to be a loop based at that point.

Definition 2.1.

Let XX be a topological space, and let x0∈Xx_{0}\in X. The set of all path homotopy classes of loops based at x0x_{0} is a group, denoted π1⁢(X,x0)\pi_{1}(X,x_{0}) and called the first fundamental group (or just the fundamental group) at x0x_{0}.

Theorem 2.2.

Let α\alpha be a path from x0x_{0} to x1x_{1} and define a map

α^:π1⁢(X,x0)→π1⁢(X,x1)\hat{\alpha}:\pi_{1}(X,x_{0})\rightarrow\pi_{1}(X,x_{1})

by

[f]↦[α]−1⋆[f]⋆[α][f]\mapsto[\alpha]^{-1}\star[f]\star[\alpha]

where [α]−1[\alpha]^{-1} is the path homotopy class of the reverse path from x1x_{1} to x0x_{0}. The map α^\hat{\alpha} is a group homomorphism. Furthermore, it is an isomorphism.

Note that the maps α^\hat{\alpha} are not canonical isomorphism, as they reqire the choice of a path α\alpha. Also, when x0=x1x_{0}=x_{1}, α^\hat{\alpha} is an inner automorphism.

Corollary 2.3.

If XX is path connected and x0,x1∈Xx_{0},x_{1}\in X, we have that π1⁢(X,x0)≅π1⁢(X,x1)\pi_{1}(X,x_{0})\cong\pi_{1}(X,x_{1}). So it makes sense to talk about the fundamental group of a path connected space.

Definition 2.4 (Simply connected).

A path connected space XX is said to be simply connected if for some x0∈Xx_{0}\in X, π1⁢(X,x0)=0\pi_{1}(X,x_{0})=0 and hence for every x∈Xx\in X.

Definition 2.5 (Pointed Top).

The category consisting of pairs (X,x0)(X,x_{0}) as objects and continuous maps f:X→Yf:X\rightarrow Y such that f⁢(x0)=y0f(x_{0})=y_{0} as morphisms between (X,x0)(X,x_{0}) to (Y,y0)(Y,y_{0}) is called the category of pointed topological sapces, denoted Top⋆{\textbf{Top}}_{\star}.

Definition 2.6 (Induced homomorphism).

Let ϕ:(X,x0)→(Y,y0)\phi:(X,x_{0})\rightarrow(Y,y_{0}) be a map of pointed topological spaces. Define the map

ϕ⋆:π1⁢(X,x0)→π1⁢(Y,y0)\phi_{\star}:\pi_{1}(X,x_{0})\rightarrow\pi_{1}(Y,y_{0})

by [f]↦[ϕ∘f][f]\mapsto[\phi\circ f]. This map is said to be the homomorphism induced by ϕ\phi.

Corollary 2.8.

If h:(X,x0)→(Y,y0)h:(X,x_{0})\rightarrow(Y,y_{0}) is an isomorphism, then h⋆:π1⁢(X,x0)→π1⁢(Y,y0)h_{\star}:\pi_{1}(X,x_{0})\rightarrow\pi_{1}(Y,y_{0}) is an isomorphism.
