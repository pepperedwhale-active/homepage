1.  The Singular Homology Group

Definition 1.1.

A pp simplex Δp⊆ℝp\Delta_{p}\subseteq\mathbb{R}^{p} is defined to be equal to the convex set spanned by {ei|i=1,2,..,p}∪{0}\{e_{i}|i=1,2,..,p\}\cup\{0\}.

Definition 1.2.

Let XX be a topological space. A singular pp simplex on XX is an continuous map X→ΔpX\rightarrow\Delta_{p}. The free abelian group generated by all singular pp simplices on XX is called the p−p-th singular chain group on XX, written Cp⁢(X)C_{p}(X), and the elements of this group are called chains.

Definition 1.3.

We define the i−i-th face of the singular pp-simplex Fi,p:Δp−1→ΔpF_{i,p}:\Delta_{p-1}\rightarrow\Delta_{p} as follows:

e0\displaystyle e_{0} ↦e0\displaystyle\mapsto e_{0}
e1\displaystyle e_{1} ↦e1\displaystyle\mapsto e_{1}
e2\displaystyle e_{2} ↦e2\displaystyle\mapsto e_{2}
⋮\displaystyle\vdots
ei−1\displaystyle e_{i-1} ↦ei−1\displaystyle\mapsto e_{i-1}
ei\displaystyle e_{i} ↦ei+1\displaystyle\mapsto e_{i+1}
ei+1\displaystyle e_{i+1} ↦ei+2\displaystyle\mapsto e_{i+2}
⋮\displaystyle\vdots
ep−1\displaystyle e_{p-1} ↦ep\displaystyle\mapsto e_{p}
Definition 1.4.

Let σ:Δp→X\sigma:\Delta_{p}\rightarrow X be a singular pp-simplex. Define a singular p−1p-1-chain called the boundary of σ\sigma by:

∂σ=∑i=0p(−1)i⁢σ∘Fi,p\partial\sigma=\sum_{i=0}^{p}(-1)^{i}\sigma\circ F_{i,p}

Note that the map ∂:Cp⁢(X)→Cp−1⁢(X)\partial:C_{p}(X)\rightarrow C_{p-1}(X) is a group homomorphism, it is called the singular boundary operator, or just the boundary operator. Sometimes, we will also denote it by ∂p\partial_{p}.

Definition 1.5.

A singular p−p-chain cc is called a p−p-cycle if we have ∂c=0\partial c=0. A singular p−p-chain cc is called a p−p-boundary if there exists a p+1−p+1-chain bb with ∂b=c\partial b=c.

Note that the set of all p−p-boundaries and cycles form groups, called Bp⁢(X),Zp⁢(X)B_{p}(X),Z_{p}(X) respectively. Furthermore, we have that ker⁢∂p=Zp⁢(X)\ker\partial_{p}=Z_{p}(X), and Im⁢(∂p+1)=Bp⁢(X)\mathrm{Im}(\partial_{p+1})=B_{p}(X).
