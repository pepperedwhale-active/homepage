3.  Covering Spaces

Definition 3.1.

Let p:E→B be a surjective map of topological spaces. The open set U of B is said to be evenly covered by B if p−1⁢(U) can be written as the disjoint union of open sets Vα such that the restriction of p to Vα is a homeomorphism onto U. The collection {Vα} will be called a partition of p−1⁢(U) into slices

Definition 3.2 (Covering space).

Let p:E→B be a surjective map of topological sapces. If for every b∈B, there is an open U⊃{b} such that U is evenly covered by p, then E is said to be a covering space for B and p is said to be a covering map .

That is to say, if B is locally evenly covered by p, then E is a covering space for B.

How do coverings relate to the fundamental group? we make this connection using lifting.

Definition 3.4.

Let p:E→B be a covering map. A lift of f:X→B along p is a map f~ such that p∘f~=f.

I  Lifting Properties of Covering Spaces

Theorem 3.5.

Let p:E→B be a covering map, and let f:[0,1]→B be a path with f⁢(0)=b. If e∈p−1⁢(b), then there is a unique lift f~:[0,1]→E with f~⁢(0)=e.

Proof.

Since p is a covering map, there exists an open cover {Uα}α∈J of B such that each Uα is evenly covered by p. Now, note that {f−1⁢(Uα)}α∈J is an open cover of [0,1], and by the lebesgue number lemma, there is a δ>0 with the property that for any x∈[0,1], we have (x,x+δ)⊆f−1⁢(Uα) for some α∈J. Hence we can find a finite subdivision 0=s0<s1<⋯<sn=1 such that f⁢([si,si+1])⊆Uα for some α∈J.

Define f~⁢(0)=e, and assume that f~ has been continuously defined on [0,si] for some i≥0. We extend f~ continuously to [0,si+1]. In order to do so, note that f⁢([si,si+1])⊆Uα for some α∈J, and p−1⁢(Uα)=⨆j∈IVj⁢⋯⁢(1) for some open Vj with p|Vj:Vj→Uα a homeomorphism for each j∈I. Pick the unique j with f~⁢(si)∈Vj. Then using (1) we have that f~⁢(s)=p|Vj−1⁢(f⁢(s)) is a continuous extension to [0,si+1] by the pasting lemma. ∎

The above theorem shows that paths lift uniquely to other paths in the covering space given the starting point of the lifted path.

Theorem 3.8.

We have the isomorphism

π1⁢(S1)≅ℤ
Proof.

The map t↦(cos⁡(2⁢π⁢t),sin⁡(2⁢π⁢t)) is a covering map, and we take the lifting correspondence ϕ:π1⁢(S1)≅π1⁢(S1,(0,1))→p−1⁢((0,1))=ℤ w.r.t 0. This map is certainly bijective, since ℝ is simply connected. We need to show that this map is a homomorphism. To that end, let ϕ⁢([f])=m,ϕ⁢([g])=n. Note that if f~,g~ are the unique lifts of f,g along p, then f~⋆(m+g~) goes from 0 to m+n, and a simple computation shows that it is a lifting of f⋆g along p. Thus ϕ⁢([f]⋆[g])=m+n=ϕ⁢([f])+ϕ⁢([g]). ∎
