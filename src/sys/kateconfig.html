Kate: How to Auto Recompile LaTeX on Saving and How to Set Up Synctex

As a KDE user, I like using the default text editor that comes with KDE, Kate. Most of what I do is typesetting documents (notes, home work, etc.), and for that I like to have a good LaTeX workflow. What I mean by this is that I want to be able to write LaTeX in one tab on one half of my screen, and have a preview of the PDF updating live on the other half. I also want to be able to do inverse synctex from my pdf to Kate. For this, it is very helpful to know how Kate plugins work and I will be showing how to achieve this workflow in this document. See the end for synctex setup.

Running Shell Scripts using Kate

We will be using this functionality to achieve what we want. First, make sure that you have a distribution of LaTeX installed. In my case, i'm using TeXLive.Now, in Kate, go to Settings -> Configure Kate... -> External Tools and click on Run Shell Script under the Tools dropdown. Then click on -> Add Tool near the bottom left of the Configure Kate window

At this point you will be greeted with a window having the fields required to make a shell script that will run upon save. We plan to use this by getting Kate to run the command
pdflatex < current_file.tex > for us, everytime we save.


To do so, set any name you want in the Name field
In Exectuable, type pdflatex
In Arguments, type %{Document:FileName}
(if you want to set up synctex, type -synctex=1 -interaction=nonstopmode %{Document:FileName} instead)
In Mime Types , type text/x-tex (This is the mime type for LaTeX documents).
In Save , select Current Document
In Trigger, select After Save
Leave everything else as it is, and click Ok .

Setting up synctex (Optional)

I will be showing how to set up reverse synctex in your favourite pdf viewer, which in my case is sioyek. First, go to your config file. In the case of sioyek, go to prefs_user.config and add the following command: inverse_search_command kate -l %2 "%1". Here "%2,%1" refers to the line and file number respectively.

Congratulations! you have now set up auto recompile on save with Kate. Now, you can use a pdf viewer like Okular or Sioyek on one half of your screen, and edit your document on the other half in Kate, and every time you save the document you are working on, Kate should automatically recompile the document and the changes should be visible in your pdf.
