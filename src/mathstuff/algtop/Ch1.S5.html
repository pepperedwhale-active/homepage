The wedge sum X1∨⋯∨Xn of topological spaces is defined as the quotient of ⨆jXj by the equivalence relation generated by p1⁢⋯~⁢p~n. . Let ⋆ be the point in X1∨⋯∨Xn that is the equivalence class of the points pi.

Definition 5.3.

a point p∈X is said to be a nondegenerate base point if there exists a neighbourhood W of p that admits a strong deformation retract onto p.

Note for example, that every point in a manifold is a nondegenerate base point.

Lemma 5.4.

Suppose that pi∈Xi are nondegenerate base points for every i. Then ⋆ is a nondegenerate base point in X1∨⋯∨Xn.

Proof.

Indeed, if Wi are neighbourhoods admitting strong deformation retracts Hi, then defining H:(⨆iWi)×I→⨆iWi induces the desired strong deformation retract. ∎

Theorem 5.5.

If X1,⋯,Xn are spaces with nondegenerate base points pi. The map

Φ:★i⁢π1⁢(Xi,pi)→π1⁢(X1∨⋯⁢Xn)

induced by the inclusion ij:Xj↪X1∨⋯∨Xn is an isomorphism.

Proof.

We prove this for the case of two spaces. We want to be able to take U=X1,V=X2, but the problem is that these are not open. So, if Wi are neighbourhood of pi admitting strong deformation retracts onto pi, we consider U1=q⁢(X1⊔W2),U2=q⁢(X2⊔W1) where q is the canonical map X1⊔X2→X1∨X2. Further, the inclusions

{⋆}↪U1∩U2⁢X1↪U1,X2↪U2

are homotopy equivalences. Furthermore, U1∩U2 is contractible hence simply connected, so that

π1⁢(U1,⋆)⋆π1⁢(U2,⋆)→π!⁢(X1∨X2,⋆)

is an isomorphism by the Van Kampen Theorem. But since i1,i2 were homotopy equivalences, we have isomorphisms π1⁢(Xi,pi)→π1⁢(Ui,⋆). By induction, we are done. ∎
