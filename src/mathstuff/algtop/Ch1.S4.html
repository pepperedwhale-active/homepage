4.  Consequences of π1⁢(S1)≅ℤ

Definition 4.1.

Let X be a topological space. For A⊆X, a retraction r:X→A is a map of topological spaces such that r∘iA=i⁢dA where iA:A↪X is the inclusion. If for a subspace A there exists such an r, we say that A is a retract of X.

Ex 4.1.

If A is a retract of X, and a0∈A then the induced map (iA)⋆:π1⁢(A,a0)→π1⁢(X,a0) is an injection. (What does it say about r⋆?)

Lemma 4.2.

Proof.

Theorem 4.3 (Brouwer’s fixed point theorem for 2 dimensions).

Let f:B2→B2 be a continuous mapping. Then f has a fixed point.

Proof.

We assume that there is a map f:B2→B2 with no fixed points for contradcition. Thus, ∀x∈B2:x≠f⁢(x), so we can pick tx>0 such that |x+(f⁢(x)−x)⁢tx|=1. Now define

h⁢(x)=f⁢(x)+tx⁢(x−f⁢(x))

and note that tx varies continuously with x, because it can be solved for using the quadratic formula. Thus h is continuous, and we also have that h⁢(x)=x for all x∈S1. Thus we have a retraction h:B2→S1, a contradcition. ∎

prove the one dimensional version of the above theorem.

I  Quotient Spaces

Definition 4.4.

Let f:X→Y be a surjective function, where X is a topological space. The quotient topology on Y induced by X,f is the collection of subsets V⊆Y such that f−1⁢(V) is open in X.

Ex 4.3.

verify that this is indeed a topology.

Definition 4.5.

a surjective function f:X→Y is said to be a quotient map if Y has the quotient topology from X,f.

Remark 2.

note that a quotient map is continuous.

Lemma 4.6.

Let f:X→Y be a function. Then the quotient topology on Y induced by X,f is the finest topology with respect to which f is continuous.

Remark 3.

A surjective function f:X→Y is a quotient map iff the following condition holds:
A subset K is closed in Y iff f−1⁢(K) is closed in X

Lemma 4.7.

if f:X→Y is a surjective open or closed map, then f is a quotient map.

Theorem 4.8.

Let f:X→Y be a quotient map. Let B⊆Y and A=f−1⁢(B)⊆X be subspaces and let g:A→B be the restricted map.

  1. 1.

    If A is open or closed, then g is a quotient map

  2. 2.

    If f is an open or closed map, then g is a quotient map.

Perhaps the most important property of quotient maps is the following:

Theorem 4.9.

Let f:X→Y be a quotient map, and let Z be a top. sp. Let h:X→Z be a function such that h⁢(x)=h⁢(x′) whenever f⁢(x)=f⁢(x′). Then h induces a unique function g:Y→Z such that h=g∘f. The function g is continuous iff h is continuous. The function g is a quotient map iff h is a quotient map.

These are not hard to prove, but they are results to keep in mind.

Corollary 4.10.

Let h:X→Z be a continuous surjective map. Let ∼ be the equivalence relation x∼y iff h⁢(x)=h⁢(y), and let X/∼. Give X/∼ the quotient topology from the canonical surjection π:X→X/∼.

  1. 1.

    the map h induces a continuous bijective map g:X/∼→Z

  2. 2.

    the above map is a homeomorphism iff h is a quotient map.

  3. 3.

    If Z is hausdorff, so is X/∼.
