Definition 4.1.

Let X be a topological space. For A⊆X, a retraction r:X→A is a map of topological spaces such that r∘iA=i⁢dA where iA:A↪X is the inclusion. If for a subspace A there exists such an r, we say that A is a retract of X.

Theorem 4.3 (Brouwer’s fixed point theorem for 2 dimensions).

Let f:B2→B2 be a continuous mapping. Then f has a fixed point.

Proof.

We assume that there is a map f:B2→B2 with no fixed points for contradcition. Thus, ∀x∈B2:x≠f⁢(x), so we can pick tx>0 such that |x+(f⁢(x)−x)⁢tx|=1. Now define

h⁢(x)=f⁢(x)+tx⁢(x−f⁢(x))

and note that tx varies continuously with x, because it can be solved for using the quadratic formula. Thus h is continuous, and we also have that h⁢(x)=x for all x∈S1. Thus we have a retraction h:B2→S1, a contradcition. ∎
