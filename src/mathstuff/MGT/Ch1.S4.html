Theorem 1.4.3.

Let

0→M⁢→𝑓⁢N⁢→𝑔⁢M′→0

be an exact sequence, and let F be any module. Then

M⊗RF⁢→f⊗i⁢d⁢N⊗RF⁢→g⊗i⁢d⁢M′⊗RF→0

is exact. I.e, tensoring preserves right exactness.

Proof.

Checking that N⊗RF→M′⊗RF is surjective is simple, since any pure tensor can be mapped to be elements of N⊗RF. Now, if we call ϕ:=f⊗i⁢d,ψ:=g⊗i⁢d, then we note that im⁡ϕ⊆ker⁡ψ. Thus it suffice to show that the induced map ψ~:N⊗RF/im⁡ϕ=coker⁢(ϕ)→M′⊗RF is injective. To that end, note that we define a left inverse as follows. Let a⊗b∈M′⊗RF. Then there is c⊗d∈N⊗RF with ψ⁢(c⊗d)=a⊗b. Define μ:M′⊗RF→coker⁢(ϕ) by a⊗b↦c⊗d¯, and note that μ∘ψ~=i⁢d. ∎

Exercises

Ex 1.4.1.

Let F be a field with characteristic p>0, and suppose that F⊆R and that there exists a∈R\F with ap∈F. Then show that R⊗FR has nilpotent elements

We will apply all the knowledge till now about projective modules, tensor algebras etc. to get some results in linear algebra. First we show that the determinant of a matrix with linearly independent columns is not a zero divisor in the following block of exercises.

Ex 1.4.2.

Show that if M is projective, then Tn⁢(M),Symn⁡(M),⋀n(M) are projective.

Ex 1.4.3.

Show that if P is a projective (resp. finitely generated projective) module, then the canonical map P→P∗∗ is injective (resp. bijective)

Ex 1.4.4.

if M is a projective R−module, then y=x1∧⋯∧xn≠0⇔∃f:⋀nM→R such that f⁢(y)=μ≠0

Ex 1.4.5.

Let M be an R−module, and let f:Mn−1→R be alternating. Then x1∧⋯∧xn=0⇔∑i=1n(−1)i⁢f⁢(x1,⋯,x^i,⋯,xn)⋅xi=0.

Ex 1.4.6.

Let M be a projective R−module. Then x1,⋯,xn are linearly dependent iff ∃0≠λ∈R:λ⁢x1∧⋯∧xn=0.

Ex 1.4.7.

Let M be a free module, and let u be any module map from M. Then u is injective ⇔det(u) is not a zero divisor in R. Alternatively, if a matrix has all linearly independent columns, its determinant is not a zero divisor in R.

The above result is quite easily obtained in linear algebra over a field, but here we have to do a little more work. This result can now be used to generalise Ex.1.1.5 to arbitrary commutative rings:

Ex 1.4.8.

Let M be a free module of rank n over a commutative ring, and N a free submodule. Then dimN≤dimM. Does this need to be true if R were not commutative?

Ex 1.4.9.

Do the same exercise as above, but use the adjugate matrix to show Ex.1.4.7.

Ex 1.4.10.

Let M be a free R−module of rank n. Show that a generating set of size n is a basis. (Hint: what happens if you wedge a linear dependence relation?)
