If di is the degree of fi and bi the leading coefficient of fi, consider the chain of ideals (b1)⊆(b1,b2)⊆⋯. This sequence stabilises, say at the m−th ideal. Thus there is an R−linear combination ∑i=1mri⁢bi=bm+1. Now consider the polynomial

Theorem 1.2.4 (Structure Theorem).
Proof.

Let R be a PID, M be an R−module. First we split M into its free and torsion parts using the following lemmas, all with alternate proofs.

Lemma 1.2.5.

Let R be a PID. Then every f.g. torsion free R−module is free.

Proof.

Let N be a f.g torsion free R−module. Let ℬ={v1,v2,⋯,vn} be a minimal cardinality generating set in N. We show that ℬ is linearly independent. Assume by induction that this is true for modules with minimal generating set size n−1. Now define the collection of submodules Λ={R⁢y:y∈N,v1∈R⁢y}. Since R is a PID, it is noetherian, so that N is noetherian. Let R⁢y′ be a maximal element of Λ. Now consider N/R⁢y′.
We claim that it is torsion free.
Indeed, suppose that for some x+R⁢y′, we have a⁢(x+R⁢y′)=0 for some a≠0. We will show that x∈R⁢y′. First, note that all such as form an ideal, so that they are generated by some 0≠a′∈R. This a′ must satisfy a′⁢x=b′⁢y′ for some b′≠0∈R. Furthermore, gcd⁡(a′,b′)=1 (Why?). Let g,h∈R be such that g⁢a′+h⁢b′=1. Then y=(g⁢a′+h⁢b′)⁢y′=a′⁢(g⁢y+h⁢x)⟹y′∈R⁢(g⁢y′+h⁢x)⟹R⁢y′=R⁢(g⁢y′+h⁢x) by maximality of y′. But also note that x=(g⁢a′+h⁢b′)⁢x=(g⁢b′⁢y′+h⁢b′⁢x)=b⁢(g⁢y′+h⁢x)⟹x∈R⁢(g⁢y′+h⁢x)=R⁢y′ thus, N/R⁢y′ is torsion free. By induction, it is free, and by projectivity we have N≅R⁢y′⊕N/R⁢y′ and this is free as it is a direct sum of two free modules. ∎

Lemma 1.2.9.

Let E be a torsion module of exponent pr⁢(r≥1) for some prime element p. Let x1∈E have period pr. Let E¯=E/(x1). Let y¯1,⋯,y¯m be independent elements of E¯. Then there exists representatives yi∈E of yi¯ with yi having the same period as yi¯, and the elements x1,y1,⋯,ym are independent.

Proof.

Let y¯∈E¯ have period pn,n≥1. Let y be a representative of y¯, then pn⁢y=c⁢ps⁢x1. where p∤c. For some s≤r. If s=r then y is the desired representative, otherwise, c⁢ps⁢x1 has period pr−s, hence y has period pn+r−s, so that n+r−s≤r⟹n≤s⟹y−c⁢ps−n⁢x1 has period n.
Let yi be representatives of y¯i having the same period as yi. Then note that ∑ci⁢yi=0⟹∑ci⁢yi¯=0⟹∀i:ci⁢yi¯=0⟹∀i:ci⁢yi=0, since ci⁢yi¯=0⟹pir∣ci where pri is the period of yi and yi has the same period as yi¯. ∎

With notation as in 1.2.9, define E¯=E/(x1). We claim that dimp(E¯p)<dimp(Ep) as R/p⁢R vectorspaces. Indeed, if w¯1,⋯,w¯m are linearly independent elements of E¯p, then by 1.2.9, we have that there is an element w of period p in (pr−1⁢x1)⊆(x1) and lifts wi of the w¯is such that w,w1,⋯,wm is independent. Thus by induction on dim(Rp)(we need to show E¯ is p−torsion for this), there exist elements x¯2.⋯,x¯s of E¯ such that they have periods pr1,⋯,pr2 respectively satisfying r2≥r2≥⋯≥rs. Again, by 1.2.9, we get the primary decomposition form.
For the uniqueness, see MSE. ∎

1.3.  Counterexamples

Ex 1.3.1.

[LAM12]Let R be a ring (not necessarily commutative). Is it true that Rm≅Rn⟹m=n?

Solution.

Not necessarily. Let V be any infinite dimensional vector space over a field k. Then if we set R=End⁢(V), we have R2≅R as right R−modules. Indeed, let ε:V→V⊕V be any isomorphism. Then we get an abelian group isomorphism λ:End⁢(V)→End⁢(V)⊕End⁢(V) defined by f↦(π1∘ε∘f,π2∘ε∘f). Indeed, this is an abelian group isomorphism as it is the image of ε under the Homk⁡(V,−) functor. To see that it is a module map, note that λ⁢(f⁢g)=(π1∘ε∘f∘g,π2∘ε∘f∘g)=(π1∘ε∘f,π2∘ε∘f)⋅g=λ⁢(f)⁢g as desired. Infact, this implies that one has Rm≅Rn for all natural m,n. ∎

Ex 1.3.3.

Is it true that the direct product of projective modules needs to be projective?

Solution.

No. Consider M=∏i∈ℕℤ. We claim that this is not projective. Suppose not for contradction. Note that since ℤ is a P.I.D, M must be free. Find an uncountable set of linearly indepenedent vectors (look at chains in 𝒫⁢(ℕ)). Thus M must have uncountable dimension. Pick a prime p∈ℤ. Let S:={m∈M:vp⁢(πn⁢(m))→∞⁢ as ⁢n→∞} where vp⁢(x) is the highest power of p dividing x∈ℤ and πn is the n−th projection map. S is basically the set of all tuples such that the highest power of p dividing the slots goes to infinity. Note that S is a submodule of M. Now consider the map M→S given by multiplication by (p,p2,⋯). This is an injection, so S must also have an uncountable linearly independent set. Finally, note that S/p⁢S has countable dimension as a ℤ/(p)−module, as it a submodule of eventually zero sequences in ℤ/(p) i.e a submodule of ⨁i∈ℕℤ/(p) but this is a contradiction, as the basis must be uncountable by what we just discussed. ∎

Theorem 1.4.3.

Let

0→M⁢→𝑓⁢N⁢→𝑔⁢M′→0

be an exact sequence, and let F be any module. Then

M⊗RF⁢→f⊗i⁢d⁢N⊗RF⁢→g⊗i⁢d⁢M′⊗RF→0

is exact. I.e, tensoring preserves right exactness.

Proof.

Checking that N⊗RF→M′⊗RF is surjective is simple, since any pure tensor can be mapped to be elements of N⊗RF. Now, if we call ϕ:=f⊗i⁢d,ψ:=g⊗i⁢d, then we note that im⁡ϕ⊆ker⁡ψ. Thus it suffice to show that the induced map ψ~:N⊗RF/im⁡ϕ=coker⁢(ϕ)→M′⊗RF is injective. To that end, note that we define a left inverse as follows. Let a⊗b∈M′⊗RF. Then there is c⊗d∈N⊗RF with ψ⁢(c⊗d)=a⊗b. Define μ:M′⊗RF→coker⁢(ϕ) by a⊗b↦c⊗d¯, and note that μ∘ψ~=i⁢d. ∎
